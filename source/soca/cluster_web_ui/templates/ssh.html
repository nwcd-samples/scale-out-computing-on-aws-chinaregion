Access the cluster using Linux/Mac


Step1: Download my private key

Step2: Modify key permission

Run chmod 600 {{user}}_soca_privatekey.pem

Step3: Connect to the cluster

Run ssh -i {{user}}_soca_privatekey.pem {{user}}@{{scheduler_ip}}

Optional: Enable KeepAlive

If you don't want your session to be automatically closed after a couple of minutes of inactivity, edit ~/.ssh/config and add:
Host {{scheduler_ip}}
ServerAliveInterval 10
ServerAliveCountMax 2
Access the cluster using Windows (PuTTY)


Step1: Download my PuTTY private key

Step2: Configure PuTTY

  • Download PuTTY
  • As hostname, enter {{scheduler_ip}}
  • Navigate to Connection > SSH > Auth and enter the path of your key ({{user}}_soca_privatekey.ppk) under your "Private Key used for Authentication"
  • Save your session
  • Click connect/open to access the cluster

    • Optional: Enable KeepAlive

    If you don't want your session to be automatically closed after a couple of minutes of inactivity, go to Connection and add "3" as "Seconds between KeepAlives"
